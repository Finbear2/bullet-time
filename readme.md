# Bullet Time

## What is it?

Small Matrix client for the Pebble watches. Allows users to easily send and view messages from the pebble watch and connect it with any matrix account on any matrix server with no required companion apps. Isn't fully featured but allows for simple reading and sending.

## Features

- Room View
  - Sorted By Recent Activity For Easy Access
- Per Room Messages
- Ability to send messages
  - Uses voice dictation
- Full message view
  - Scrollable
- Web Config
